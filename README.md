# Android Music Player

## Overview
This project is an Android Music Player application developed using Android Studio. The application retrieves a playlist from a remote web server, parses XML responses, and streams audio files for playback.

## Features
- Retrieve playlist from a remote server
- Parse XML playlist data
- Display songs using RecyclerView
- Stream audio from remote URLs
- Play and pause music
- SeekBar for playback progress
- Automatic playback of the next song
- Background music playback

## Technologies Used
- Android Studio
- Java
- XML
- RecyclerView
- MediaPlayer
- HTTP Networking

## Project Structure
- RecyclerView for displaying songs
- MediaPlayer for audio playback
- XML parser for processing playlist data
- Android UI components

## How to Run
1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the application on an Android emulator or physical device.
